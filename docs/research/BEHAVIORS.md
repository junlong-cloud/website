# Behaviors

- Header stays fixed and active anchor uses cyan fill.
- Buttons shift 2px and deepen their hard shadow on hover.
- Phone and gallery imagery uses horizontally clipped strips; mobile remains horizontally scrollable.
- Section anchors scroll smoothly.
- Decorative stickers use gentle float/tilt keyframes.
- Layout switches to stacked content below 760px; desktop max width is 1440px.
