# AppWorks Specification

- Target: `src/components/AppWorks.tsx`
- Screenshot: `docs/design-references/original-section-0.png`
- Interaction: horizontal overflow for images on mobile; images rise 8px on hover.
- Intro has huge centered two-line pixel heading, hand-written kicker and rotated yellow note.
- Three projects: Hush / Indus / Founder App. Metadata row has icon+category+title at left and short description at right.
- Phone cards are pale gray rounded 28px, 220-260px wide, image aligned to top. Five repeated/cycled phones per strip.
- Desktop generous vertical spacing (~180px intro, 100px between projects); mobile uses 48px.
